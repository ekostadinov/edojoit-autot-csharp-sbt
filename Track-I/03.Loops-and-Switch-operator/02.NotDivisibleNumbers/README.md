# Not Divisible Numbers

## Description
Write a program that reads from the console a positive integer N and prints all the numbers from 1 to N not divisible by 3 or 7, on a single line, separated by a space.

|     Input      |     Output     |
|----------------|----------------|
|10              |1 2 4 5 8 10    |
|3               |1 2             |

## Constraints
- Time limit: **0.1s**
- Memory limit: **16MB**
