# Matrix of Numbers

## Description
Write a program that reads from the console a positive 
integer number N and prints a matrix like in the examples below. Use two nested loops.
  - Challenge: achieve the same effect without nested loops.

|     Input      |     Output     |
|----------------|----------------|
|2               |12              |
|                |34              |
|4               |1 2 3 4         |
|                |2 3 4 5         |
|                |3 4 5 6         |
|                |4 5 6 7         |


## Constraints
- Time limit: **0.1s**
- Memory limit: **16MB**
