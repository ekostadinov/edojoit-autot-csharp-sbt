# Calculate Factorials

## Description
Write a program that calculates `N! / K!` for given `N` and `K`. 
  - Challenge: Use only one loop for both [factorial](https://en.wikipedia.org/wiki/Factorial) calculations

|     Input      |     Output     |
|----------------|----------------|
|8 3             |6720             |


## Constraints
- Time limit: **0.1s**
- Memory limit: **16MB**
