# Allocate array

## Description
Write a program that allocates array of N integers, initializes each element by its index multiplied by 5 and the prints the obtained array on the console.

|     Input      |     Output     |
|----------------|----------------|
|5               |0             |
|          |       5   |
|          |        10  |
|          |          15|

## Constraints
- Time limit: **0.1s**
- Memory limit: **16MB**
