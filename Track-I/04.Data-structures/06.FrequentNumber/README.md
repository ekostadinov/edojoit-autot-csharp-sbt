# Frequent number

## Description
Write a program that finds the most frequent number in an array of N element.

|     Input      |     Output     |
|----------------|----------------|
|13 4 1 1 4 2 3 4 4 1 2 4 9 3             |4 (5 times)             |


## Constraints
- Time limit: **0.1s**
- Memory limit: **16MB**
